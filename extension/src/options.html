How it works
1
On save, the extension fetches /x.json from your server to build a streamer whitelist — only matching tabs will activate.
2
Open an X livestream matching a whitelisted handle. Chat is captured using WebSocket frames, fetch() responses, and DOM mutation — whichever fires first.
3
Messages are batched at the configured interval and pushed to /api/x/chat with the API key in the request header.
4
The dashboard merges X messages into the unified chat feed alongside Twitch and Kick in real time.
5
The popup shows live message counts and warns you if X's chat format changes unexpectedly.
